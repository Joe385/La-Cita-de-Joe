Use CSS Selectors to Style Elements

Create style blocks to make CSS selectors for specific elements so you dont have to use the style attribute frequently
<style>
 h2{
  color: red;
 }
</style>

Use a CSS Class to Style an Element

Classes are reusable styles that can be added to html elements and start with a . (period)
<style>
 .blue-text { (bolded for emphasis on class)
  color: blue;
 }
</style>
Add a class attribute to apply style on elements/objects
<h2 class="blue-text">Example<h2>

Change the Font Size of an Element

There are multiple attributes you can assign to a class style element. For example

h1 {
 font-size: 30px;
}

can change the font size.
However, you can also change fonts, colors, styles, etc.
