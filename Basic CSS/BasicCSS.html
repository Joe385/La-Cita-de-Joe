Use Clockwise Notation to Specify the Padding of an Element

You specify padding in one line like this

1

It's organized by top, right, bottom, and left, and hence Clockwise Notation.

1 
margin: 10px 20px 10px 20px;

Use Attibute Selectors to Style Elements

Other ways to style custom group elements is through Attribute selectors

1
2
3		 
[type='radio'] {
    margin: 20px 0px 20px 0px;
}

An attribute selector ([attr=value]) matches and styles elements that have a specific attribute value.
For example in the code above, it will find elements that have the attribute [type='radio'] and give it the custom margins.

Understand Absolute versus Relative Units

CSS has a lot of other measurements to use however, the two main types are absolute and relative.
Absolute units like in, mm, and px approximate the actual measurement on-screen.
Relative unlits like em, rem, and % are relative to another length's value. For example, em is based on the size of an element's font. So if you set a font-size property of a child, it will be relative to its parent's font-size
Although briefly explained, it will be further covered in Responsive Web Design Principles section.
